Pathfinder
==========

Small library using A* and BFS for finding shortest paths on binary maps.

How to run
==========

From commandline type:
'make update'
'make'

'./build/bin/main_astar' or './build/bin/main_bfs'

Dependencies
============

Needs:
c++11-enabled GNU toolchain, CMake, Eigen3 installed onto the system.

Notes
=====
